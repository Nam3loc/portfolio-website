Bloons Tower Defense

This game was built using HTML and HTML Canvas, CSS, and vanilla JavaScript. Bloons is a tower defense game is a game in which enemies have to cross a map and get to your tower, where they will try to destroy your tower. The goal of the game is to place towers down around the enemies path so that the towers can kill the enemies before they reach the other end of the map.

I plan to fix the glitches of the monkey tower throwing the darts and the darts hitting the balloon. I also plan to add more balloons with different characteristics like speed and layers to add depth to the game.
